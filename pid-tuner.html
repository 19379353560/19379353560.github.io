Blackbox Tool

INAV PID Tuner turns Blackbox logs into cautious tuning suggestions.

A FastAPI web tool for extracting PID error, D-term noise, and gyro-noise metrics from INAV Blackbox logs.

Back to project matrix

What Needs Feedback

  • Anonymized INAV `.bbl` and `.bfl` logs from different airframes.
  • Before/after logs after PID or filter changes.
  • Better rules for fixed wing, 5-inch, 7-inch, and long-range builds.
  • UI improvements that make generated recommendations safer to review.

Privacy Note

Remove private GPS or home-position data before sharing logs publicly. Screenshots or summary metrics are useful when full logs cannot be shared.
