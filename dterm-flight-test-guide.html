Flight Test

Collect comparable logs for INAV D-term pre-diff LPF testing.

Keep the test conservative, change one setting at a time, and stop if motors heat up or behavior changes unexpectedly.

Suggested Sequence

  • Record a baseline with `dterm_lpf2_hz = 0`.
  • Repeat the same conservative flight pattern with 200Hz.
  • Repeat only if safe with 250Hz.
  • Capture Blackbox logs and motor temperature notes after each run.

Report Back

  • Aircraft size, props, motors, battery, firmware commit, and tune profile.
  • D-term noise, gyro noise, and motor-temperature observations.
  • Any unexpected behavior when changing PID/filter profiles.
